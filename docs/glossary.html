A glossary of terms used in MeV and in this documentation.

Part -A segment of the sequence.

Strip - Component of a window used to display and manipulate musical events. For example velocity, pitch bend etc.

Piano Roll - An interface that allows the display and input of notes. An example can be seen on the top of this Image.

Destination - endpoints in your system that live outside of MeV, like the virtual MIDI port of a software synthesizer or a physical MIDI output

MIDI - The strip that allows for seqencing of parts.

Mixer - This is an interface for the management of destinations.

Arrangement - The strip that allows for seqencing of parts.
